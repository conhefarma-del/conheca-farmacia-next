 Resumo da Implementação

  A secção "Referências" foi adicionada ao ficheiro artigo.html com sucesso. Aqui estão as alterações feitas:                                                           
  1. HTML (artigo.html)                                                                                                                                                 
  - Adicionada a secção "Referências" entre o conteúdo do artigo e a secção "Artigos Relacionados"
  - A secção tem display: none por padrão e é ativada via JavaScript quando há referências

  2. JavaScript (src/article-detail.js)

  - Adicionada a função renderReferences(article) que:
    - Verifica se o artigo tem referências
    - Renderiza a lista de referências formatadas
    - Mostra/oculta a secção conforme necessário
  - A função é chamada após o carregamento dos dados do artigo

  3. CSS (src/input.css)

  - Adicionados estilos para:
    - .article-references-section - secção de referências
    - .article-references-list - lista de referências
    - .reference-item - cada referência individual
    - .reference-number - número da referência
    - .reference-text - texto da referência

  4. Dados (src/content/articles-catalog.json)

  - Adicionado campo references ao primeiro artigo como exemplo
  - O campo é um array de strings com as referências formatadas

  Como usar:

  Para adicionar referências a outros artigos, basta adicionar o campo references no JSON:
  "references": [
    "Referência 1",
    "Referência 2",
    "Referência 3"
  ]
