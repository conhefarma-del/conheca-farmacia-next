Conheça Farmácia

Bem-vindo ao portal de educação e gestão clínica.

⚠️ IMPORTANTE: Este projeto usa Vite e requer um servidor para funcionar corretamente. O CSS e JavaScript só carregam através do servidor de desenvolvimento.

🚀 Como Iniciar:

  1. Abra o terminal na pasta do projeto
  2. Execute: npm run dev
  3. Aceda a: http://localhost:5173

Alternativa: Se preferir, pode usar o ficheiro index.html através do servidor Vite (nunca diretamente).

Para mais informações, consulte DESENVOLVIMENTO.md.
